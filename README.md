# フォルダ・ファイル構成

### ルートディレクトリ
・**docker-compose.yml**: Dockerのコンテナ構成を定義します。  
・**Dockerfile**: PHP,Apache等の実行環境の構成を定義します。  
・**init.sql**: テーブル作成やメニューのデータ挿入をします。  
・**README.md**: フォルダ・ファイル構成について説明します。  

### src/（プログラム）
 ・**index.php**: トップページ。画像マップやコメント一覧、ランキングやコメントページへ遷移できます。  
 ・**detail.php**: 牛肉の各部位の詳細ページ。部位ごとにコメント投稿・閲覧ができます。  
 ・**comment.php**: コメント専用ページ。メニュー選択で店舗全体や各部位を選んだり、名前や年代、コメントを選択したりして投稿できます。  
 ・**submit_comment.php**: 投稿したコメントをDBに保存して完了後に元のページへリダイレクトします。  
 ・**ranking.php**: ランキング表示・投票ページ。投票によってメニューのランキングを図に表示させます。ランキング表示は順位順です。  
 ・**db.php**: データベースへの接続設定の管理をするファイル。  
 ・**get_ranking_json.php**: ランキングデータをJSON形式で返します。Ajax通信で使用します。  
 ・**submit_vote_ajax.php**: 投票を受け付けてデータベースを更新します。  

### src/css（デザイン）
・**style.css**: 背景やボタン、画像マップ等のデザインを定義しています。  

### src/image（画像）
・**cuts of meat.jpg**: 画像マップで用いる画像。牛の部位が書かれています。  
・**outside.jpg**: 全体のページの背景画像。文字が見えやすいようにデザインで暗くしています。
